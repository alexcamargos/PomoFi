# Como rodar o projeto

Este projeto foi testado e verificado.

## Passos para rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Para rodar o servidor de desenvolvimento:
   ```bash
   npm start
   ```
   O servidor iniciará em `http://localhost:4200/`.

3. Para fazer o build de produção:
   ```bash
   npm run build
   ```
   Os arquivos serão gerados na pasta `dist/`.
